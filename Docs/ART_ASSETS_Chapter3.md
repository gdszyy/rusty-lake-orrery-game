# 第三章美术资产需求清单

**章节名称**: 时间的交响  
**主题**: 流动的永恒与创造的未来  
**符号**: 未来之符（齿轮+星辰+画笔的融合）  
**场景数量**: 1个核心大厅（4个功能区域）  
**最后更新**: 2025-12-08

---

## 📐 场景结构概述

第三章采用宽屏场景设计（2560x1080），分为四个功能区域：

- **列宾画作展墙**（左侧）：艺术与永恒主题
- **天体仪核心区域**（中央）：时间机械主题
- **游戏角落**（右侧）：娃娃机与彩票兑换机
- **工作台区域**（下方）：日记、调色盘、画笔等道具

## 🎨 十、美术风格指南

### 10.1 色彩方案

**整体色调**：
- 主色调：深邃星空蓝（#0D47A1）、典雅博物馆棕（#5D4037）
- 天体仪：金色（#FFD700）、银色（#C0C0C0）
- 画作区域：古典油画色调（深棕、暗红、金黄）
- 游戏角落：复古霓虹（#FF1744、#00E5FF、#FFEA00）
- 强调色：彩虹色（用于调色盘、画笔、永恒之符）

**符号色彩**：
- 过去之符：铜色（#B87333）
- 现在之符：红蓝交织（#F44336 + #2196F3）
- 未来之符：彩虹色（渐变）
- 永恒之符：流动彩虹光（动态渐变）

### 10.2 光影风格

- **天体仪区域**：神秘的星空光，金银色机械光泽

- **最终仪式**：金色光芒照亮整个展厅，彩虹光效

### 10.3 材质风格

- **天体仪**：金属质感，高光反射，精密机械感
- **画作**：油画质感，厚重的笔触，古典画框

## 📝 十二、特殊注意事项

### 12.1 娃娃机制作注意事项

1. **双人操作机制**：
   - 需要清晰的视觉反馈区分左右摇杆
   - 分屏效果需要流畅，不能有卡顿
   - 爪子的移动需要有物理感

2. **抓取动画**：
   - 爪子下降→抓取→上升→移动→掉落，需要5-7帧
   - 扭蛋掉落需要有重量感
   - 成功时的"You Win!"特效要醒目

### 12.2 画作制作注意事项

1. **列宾风格**：
   - 参考列宾的古典油画风格（厚重、光影对比强烈）
   - 画作1和2需要能识别出第一、二章的场景元素
   - 画作3的动态效果是重点，需要精美的粒子效果

2. **绘画小游戏**：
   - 星辰连接的轨迹需要流畅
   - 彩虹画笔光迹需要有艺术感
   - 完成后的符号需要清晰可辨

### 12.3 天体仪制作注意事项

1. **机械感**：
   - 齿轮旋转需要精确对齐
   - 365个刻度的点亮需要有序列感
   - 金属材质需要高质量的反射效果

2. **时钟交互**：
   - 时针分针的拖动需要流畅
   - 秒针的走动需要精确（每秒一格）
   - 午夜钟声的震动效果要有仪式感

### 12.4 最终仪式注意事项

1. **符号融合**：
   - 三个符号飞出、旋转、融合的动画需要流畅
   - 永恒之符的诞生需要震撼的视觉效果
   - 金色光芒照亮展厅需要有层次感

2. **午夜钟声**：
   - 12次钟声需要有节奏感（每次3-5秒间隔）
   - 每次钟声的文字需要清晰可读
   - 震动效果需要配合音效

### 12.5 文件命名规范

- 场景背景：`BG_[场景名]_[描述]`
- 画作：`ART_[画作名]`
- 天体仪组件：`OBJ_Orrery_[组件名]` / `OBJ_Clock_[组件名]`
- 娃娃机组件：`OBJ_ClawMachine_[组件名]`
- 兑换机组件：`OBJ_PrizeMachine_[组件名]`
- 物品栏物品：`ITEM_[物品名]`
- 特效：`VFX_[特效名]`
- UI：`UI_[UI名]`

## 🎬 十四、关键动画序列

### 14.1 娃娃机抓取序列

1. 投币 → 机器激活（霓虹灯亮起）
2. 双人操作 → 爪子移动（左右摇杆控制）
3. 按下按钮 → 爪子下降
4. 抓取扭蛋 → 爪子上升
5. 移动到出口 → 扭蛋掉落
6. "You Win!" → 彩票飞出

**总时长**：约15-20秒

### 14.2 画作创作序列

1. 放置调色盘和画笔 → 进入绘画界面
2. 星辰轮廓浮现 → 玩家连接星辰
3. 彩虹画笔光迹 → 形成符号
4. 画作完成 → 变为流动星空
5. 两个人影共舞 → 铭牌发光
6. 未来之符飞出 → 进入物品栏

**总时长**：约30-40秒

### 14.3 最终仪式序列

1. 设定时间1:47 → 大钟共鸣
2. 放置三个符号 → 刻度依次点亮
3. 天体仪旋转 → 指针推进到午夜
4. 午夜钟声（12次） → 每次钟声+独白+震动
5. 三个符号飞出 → 旋转、交织、融合
6. 永恒之符诞生 → 金色光芒照亮展厅
7. 最终独白 → "打开你面前的盒子"

**总时长**：约90-120秒

## 🎁 十五、与现实的连接

### 15.1 最终提示UI设计

**"打开你面前的盒子"**提示需要特别设计：

- 文字需要柔和的光芒效果
- 背景需要虚化，突出文字
- 可以有轻微的闪烁效果，吸引注意
- 字体需要优雅、易读
- 建议使用衬线字体，增加仪式感

### 15.2 过渡设计

从游戏到现实的过渡需要平滑：

- 屏幕逐渐淡出到柔和的光芒
- 背景音乐逐渐减弱
- 留出足够的时间让玩家沉浸在情感中
- 不要突然结束，要有余韵
