## 🎨 设计核心理念

本UI设计的核心是**将UI无缝融入游戏世界**，追求以下四个关键特质：

### 神秘 (Mysterious)
界面元素充满符号感和未知感，引导玩家探索而非直接告知。每一个图标、每一个纹理都应该让玩家感受到炼金术的神秘氛围。

### 复古 (Analog)
避免现代数字感，追求手绘、机械和炼金术手册的质感。所有UI元素都应该看起来像是从中世纪炼金术士的工作台上取下来的真实物品。

### 触感 (Tactile)
交互应感觉像在触摸真实的物理对象。拨动开关时应有阻力感，翻阅书页时应有纸张摩擦声，拾取物品时应有重量感。

### 简约 (Minimalist)
在主游戏场景中保持界面元素的最小化，最大化玩家的沉浸感。只在必要时显示UI元素，其余时间让玩家完全沉浸在游戏世界中。

## 🎮 核心UI模块概览

### 1. 主游戏界面 (HUD)
极简设计，默认几乎无UI元素，仅在交互时显示提示。右下角的皮包图标和右上角的齿轮图标是唯一的常驻元素。

### 2. 背包系统
锈湖经典的横向物品栏，采用皮革和羊皮纸材质，支持物品拾取、观察、使用和组合。

### 3. 对话与独白界面
底部半透明文本框，打字机效果逐字显示，配合沙沙的笔尖声，营造阅读炼金术日记的氛围。

### 4. 谜题界面
UI即谜题本身，所有交互元素都是谜题的一部分，无额外UI层，最大化沉浸感。

### 5. 菜单系统
设计为炼金术士的工作台，所有菜单选项都是场景中的3D对象，充满仪式感。

## 📐 技术参数速查

### 基准分辨率
- **1920 x 1080** (16:9)
- 适配范围: 1280x720 至 2560x1440

### 关键尺寸
- **按钮**: 80 x 80 像素（最小触控区域）
- **物品槽**: 120 x 120 像素
- **物品图标**: 100 x 100 像素
- **交互图标**: 64 x 64 像素

### 色彩方案
- **主色调**: #3D2B1F (深褐色), #EAE0C8 (羊皮纸色), #1A2930 (普鲁士蓝)
- **强调色**: #C9A86B (暗金色), #8C1818 (深红色)

### 字体
- **正文**: Cormorant Garamond, 32px
- **标题**: Uncial Antiqua, 手写风格

## 📝 设计原则

### 1. 叙事优先
每一个UI元素都应服务于游戏叙事，推动故事发展或揭示世界观。UI不仅是功能性的，更是叙事性的。

### 2. 反馈丰富
每一次交互都应有明确的视觉、听觉和（在移动端）触觉反馈，让玩家感受到操作的真实性。

### 3. 渐进引导
通过UI的渐进显示和提示系统，引导玩家逐步掌握游戏机制，而非一次性抛出所有信息。

### 4. 性能优先
在保证视觉效果的前提下，始终将性能放在首位，确保游戏在中低端Android设备上流畅运行。

---

## 🎯 质量标准

### 视觉质量
- 所有UI元素应符合锈湖系列的超现实主义风格
- 手绘风格应保持一致性
- 色彩方案应严格遵循设计指南

### 交互质量
- 所有交互应在0.2秒内给出反馈
- 触控区域应不小于80x80像素
- 动画应流畅，帧率不低于30fps

### 技术质量
- UI Draw Call应控制在20以内
- 纹理内存占用应不超过50MB
- UI加载时间应不超过0.5秒
