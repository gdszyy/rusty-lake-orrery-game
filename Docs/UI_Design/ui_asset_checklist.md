## 10. 资源命名规范

### 10.1 命名前缀

- **T_**: 纹理 (Texture)
- **M_**: 材质 (Material)
- **SM_**: 静态网格 (Static Mesh)
- **SFX_**: 音效 (Sound Effect)
- **AMB_**: 环境音 (Ambient Sound)
- **Anim_**: 动画 (Animation)
- **WBP_**: UMG控件蓝图 (Widget Blueprint)
- **Font_**: 字体 (Font)

### 10.2 命名规则

- 使用帕斯卡命名法（PascalCase）
- 名称应清晰描述资源内容
- 避免使用缩写（除非是通用缩写如UI、VFX）
- 同一系列资源应使用一致的命名模式

**示例**:
- `T_Item_Ring` (正确)
- `T_ring` (错误，缺少前缀说明)
- `T_Item_Rng` (错误，使用了不明确的缩写)

## 12. 资源检查清单

在提交资源前，请确保：

- [ ] 所有资源文件命名符合规范
- [ ] 纹理尺寸为2的幂次方（如64, 128, 256, 512）
- [ ] PNG图标包含正确的Alpha通道
- [ ] 音效文件为WAV格式，44.1kHz采样率
- [ ] 3D模型面数在限制范围内
- [ ] 所有资源已放置在正确的文件夹路径
- [ ] 资源文件大小已优化（纹理压缩、音频压缩）
- [ ] 手绘风格一致性检查
- [ ] 色彩方案符合设计指南
