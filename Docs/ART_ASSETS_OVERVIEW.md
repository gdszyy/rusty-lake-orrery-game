## 🎯 项目概述

### 游戏结构

本游戏共分为三章，每章一个主场景：

| 章节 | 场景名称 | 主题 | 场景分辨率 | 区域数量 |
|-----|---------|------|-----------|---------|
| 第一章 | 博物馆第一展厅 | 凝固的美丽与永生的诅咒 | 1920x1080 | 1个展厅 |
| 第二章 | 博物馆第二展厅 | 植物、镜像与成长 | 1920x1080 | 2个区域 |
| 第三章 | 博物馆核心大厅 | 时间的交响 | 2560x1080 | 4个区域 |

### 美术风格定位

**核心风格**：锈湖系列的标志性美术风格
- 神秘而精致的手绘质感
- 强烈的光影对比
- 超现实主义的元素融合
- 炼金术符号的视觉语言
- 机械美学与自然元素的结合

**色彩基调**：
- 第一章：暖金色（珠宝）+ 深褐色（埃及）+ 幽蓝色（夜晚）
- 第二章：翠绿色（植物）+ 冷蓝色（镜像）+ 彩虹色（花朵）
- 第三章：星空蓝（天体仪）+ 博物馆棕（典雅）+ 霓虹色（游戏角落）

## 🎨 第二章美术资产概览

### 场景结构

**博物馆第二展厅：生长的悖论**（1920x1080）

```
┌─────────────────────────────────────────────────────────────┐
│          博物馆第二展厅：生长的悖论                          │
│                                                             │
│  ┌──────温室区域──────┐      ┌──────镜像区域──────┐       │
│  │ [天窗·可开关]      │      │ [大镜子]           │       │
│  │ [花盆1-4]          │      │ [红色人偶]         │       │
│  │ [浇水壶] [时钟]    │      │ [蓝色人偶]         │       │
│  │ [园艺笔记]         │      │ [按钮1] [按钮2]    │       │
│  └────────────────────┘      └────────────────────┘       │
│              [中央：蝴蝶飞舞区域]                           │
└─────────────────────────────────────────────────────────────┘
```

### 核心资产亮点

1. **植物生长动画**（4种×5帧）：
   - 向日葵（金色，阳光植物）
   - 睡莲（蓝色，水分植物）
   - 月见草（紫色，时间植物，夜晚发光）
   - 蔷薇（粉红色，独立植物）

2. **镜像人偶系统**：
   - 红色人偶（代表"你"）
   - 蓝色人偶（代表"TA"，镜像行动）
   - 紫色融合人偶（红+蓝融合）

3. **蝴蝶骨骼动画**：
   - 彩色蝴蝶，翅膀有时钟图案
   - 飞行轨迹连接4朵花和2个人偶
   - 形成"时间之网"

4. **核心符号**：
   - 红色花朵（从镜子碎片中浮现）
   - 蓝色花朵（从时间之网凝固）
   - 现在之符（两朵交织的花，红蓝相间）

### 关键特效

- 植物生长光效
- 花朵绽放特效
- 花瓣飘落
- 镜子破裂
- 人偶融合光效
- 时间之网（光线网络）
- 符号形成

## 🎨 第三章美术资产概览

### 场景结构

**博物馆核心大厅：时间的交响**（2560x1080）

```
┌───────────────────────────────────────────────────────────────────────────────────────────┐
│                                博物馆核心大厅：时间的交响                                 │
│                                                                                           │
│      [列宾画作展墙]                     [天体仪核心]                    [游戏角落]        │
│ [画1] [画2] [画3]                    [大钟+符号槽]                [娃娃机] [兑换机]      │
│                                                                                           │
│ [工作台] [日记] [调色盘]                                                                 │
└───────────────────────────────────────────────────────────────────────────────────────────┘
```

### 核心资产亮点

1. **列宾画作系统**：
   - 画作1：凝固的过去（第一章场景）
   - 画作2：分离的现在（第二章场景）
   - 画作3：流动的未来（动态星空，玩家创作）

2. **天体仪机械系统**：
   - 365个刻度的外环
   - 大钟（时针、分针、秒针）
   - 3个符号槽（过去、现在、未来）
   - 3个齿轮槽（铜、银、金）

3. **娃娃机系统**：
   - 复古娃娃机外壳（霓虹灯装饰）
   - 机械爪（可双人操作）
   - 左右摇杆
   - 3个齿轮扭蛋（铜、银、金）

4. **彩票兑换机系统**：
   - 复古兑换机外壳
   - 奖品列表屏幕
   - 永恒调色盘（头奖）

5. **核心符号**：
   - 过去之符（乌鸦羽毛，铜色）
   - 现在之符（交织的花，红蓝色）
   - 未来之符（彩虹画笔）
   - 永恒之符（三者融合，流动彩虹光）

### 关键特效

- 双人操作分屏效果
- 扭蛋打开光芒
- 彩票飞舞
- 颜料流动形成符号
- 画作变为流动星空窗户
- 星河中两个人影共舞
- 365个刻度依次点亮
- 午夜钟声光环（12次）
- 符号飞出、旋转、融合
- 永恒之符诞生光芒

## 📐 技术规范

### 文件格式

| 资产类型 | 源文件格式 | 导出格式 | 备注 |
|---------|-----------|---------|------|
| 场景背景 | PSD | PNG | 保留图层 |
| 可交互物品 | PSD | PNG | 需要透明通道 |
| 角色/人偶 | PSD | PNG序列 | 每帧独立文件 |
| 植物动画 | PSD | PNG序列 | 5帧动画 |
| 物品栏图标 | PSD | PNG | 128x128 |
| UI元素 | PSD | PNG | 保留透明度 |
| 特效 | - | 粒子系统/Shader | 引擎内制作 |

### 分辨率规范

| 资产类型 | 推荐分辨率 | 最大文件大小 |
|---------|-----------|-------------|
| 场景背景 | 1920x1080 或 2560x1080 | 3MB |
| 大型物品 | 512x512 或更大 | 2MB |
| 中型物品 | 256x256 | 1MB |
| 小型物品 | 128x128 | 500KB |
| 物品栏图标 | 128x128 | 200KB |
| UI元素 | 可变 | 1MB |

### 命名规范

**统一命名格式**：`[类型前缀]_[章节]_[描述]_[状态/帧数]`

**类型前缀**：
- `BG_` - 背景
- `OBJ_` - 可交互物品
- `CHAR_` - 角色/人偶
- `ITEM_` - 物品栏物品
- `VFX_` - 特效
- `UI_` - UI元素
- `DECO_` - 装饰元素
- `TEX_` - 纹理
- `ANIM_` - 动画序列

**示例**：
- `BG_Chapter2_Greenhouse_Full.png`
- `OBJ_Chapter2_FlowerPot_01.png`
- `CHAR_Chapter2_RedPuppet_Idle_01.png`
- `ITEM_C2_001_Sunflower_Seed.png`
- `VFX_Chapter2_Plant_Grow`

## 🎨 美术风格参考

### 锈湖系列风格要素

1. **手绘质感**：
   - 保留手绘的笔触和纹理
   - 避免过于光滑的矢量感
   - 适度的颗粒感和噪点

2. **光影对比**：
   - 强烈的明暗对比
   - 聚光灯效果
   - 神秘的阴影

3. **色彩运用**：
   - 低饱和度的主色调
   - 高饱和度的强调色
   - 金色、铜色、银色的金属光泽

4. **超现实元素**：
   - 机械与自然的融合
   - 炼金术符号
   - 时间与空间的视觉化

5. **细节丰富**：
   - 纹理细节（木纹、金属、石材）
   - 装饰细节（花纹、刻痕、铭文）
   - 动态细节（光效、粒子、动画）

### 第二章风格关键词

- **温室区域**：自然、温暖、生机、柔和
- **镜像区域**：冷硬、机械、对称、反射
- **整体氛围**：成长、分离、独立、连接

### 第三章风格关键词

- **天体仪区域**：机械、精密、庄严、永恒
- **画作区域**：古典、艺术、厚重、流动
- **游戏角落**：复古、霓虹、街机、欢乐
- **整体氛围**：仪式感、情感高潮、虚实交融
