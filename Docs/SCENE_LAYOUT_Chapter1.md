## 🏰 钟楼整体设计

### 建筑风格

**时代**: 19世纪维多利亚时期  
**风格**: 哥特式 + 炼金术实验室  
**状态**: 废弃、停滞、诡异

- **材质**: 深灰色石砖，部分长满苔藓

- **塔顶**: 尖顶，顶部有大钟和乌鸦

### 详细布局

#### 1.1 入口区域 (南侧)

**大门**:
- 位置: 南墙中央
- 类型: 双开木门，厚重，有铁质装饰
- 状态: 半开（玩家从此进入）
- 门上雕刻: 炼金术符号和时钟图案
- 音效: 开门时发出沉重的"吱呀"声

**门厅**:
- 地面: 黑白相间的大理石地砖（棋盘格）
- 墙壁: 深色木质护墙板
- 天花板: 拱顶，中央有吊灯（不亮）

#### 1.2 齿轮装置 (西北角)

**位置**: 距离北墙2米，距离西墙2米

**装置描述**:
- 一个大型的黄铜机械装置
- 高度: 2米
- 宽度: 1.5米
- 三个空轴位，分别标记着符文：
  - 左侧小轴: "日常" (古代文字)
  - 中间中轴: "回忆" (古代文字)
  - 右侧大轴: "未来" (古代文字)

**装置细节**:
- 底座: 厚重的石质底座，刻有炼金术圆环
- 框架: 黄铜框架，有精美的雕刻
- 内部: 可以看到复杂的齿轮系统（静止）
- 抽屉: 底部有隐藏的抽屉（初始关闭）

**周围环境**:
- 地面有油渍和金属碎屑
- 墙上挂着工具：扳手、锤子、量规
- 一张工作台，上面有设计图纸

**光照**:
- 装置上方有一盏聚光灯（微弱）
- 符文在黑暗中微微发光

#### 1.3 墙上的画作 (北墙)

**位置**: 北墙，三幅画并排

**画作1 (左侧)**:
- 内容: 一个炼金术士在实验室工作
- 风格: 写实主义，但色调阴郁
- 细节: 炼金术士的脸模糊不清

**画作2 (中间)**:
- 内容: 一只巨大的乌鸦栖息在钟楼上
- 风格: 超现实主义
- 细节: 乌鸦的眼睛似乎在注视观者

**画作3 (右侧)**:
- 内容: 一座植物园，阳光明媚
- 风格: 印象派
- 细节: 画面中有两个模糊的人影
- **特殊**: 这幅画似乎在"呼吸"，植物微微摇曳

**画框**:
- 材质: 厚重的木质画框，有金色装饰
- 状态: 部分画框有裂痕

#### 1.4 长椅与日记 (东南角)

**长椅**:
- 位置: 东墙边，靠近楼梯
- 类型: 木质长椅，有靠背
- 状态: 陈旧，表面有划痕
- 细节: 椅背上刻着日期："1869"

**日记第一页**:
- 位置: 长椅上，打开着
- 外观: 泛黄的纸页，古老的墨水
- 状态: 可以拾取阅读

#### 1.5 螺旋楼梯 (中央偏东)

**位置**: 大厅中央偏东

**楼梯描述**:
- 类型: 铁质螺旋楼梯
- 旋转方向: 顺时针向上
- 栏杆: 铁质，有锈迹，栏杆上有藤蔓缠绕
- 台阶: 每级台阶约20cm高

**楼梯音效**:
- 脚步声: "咯吱咯吱"的金属声
- 栏杆: 触摸时发出"嘎吱"声

### 美术风格

#### 色调

- **主色调**: 深灰、棕褐、暗金
- **强调色**: 符文的微弱蓝光、画作的色彩
- **整体氛围**: 阴郁、停滞、神秘

#### 材质

| 对象 | 材质 | 特征 |
|------|------|------|
| 地面 | 大理石 | 黑白棋盘格，部分有裂痕 |
| 墙壁 | 石砖 + 木护墙板 | 深色，有苔藓和霉斑 |
| 天花板 | 石质拱顶 | 有裂纹，蜘蛛网 |
| 齿轮装置 | 黄铜 + 石材 | 氧化的黄铜，沉重的石座 |
| 画框 | 木质 + 金箔 | 陈旧，金箔剥落 |
| 长椅 | 木质 | 深色，有划痕和磨损 |

#### 细节装饰

- **蜘蛛网**: 角落和天花板
- **灰尘**: 所有表面都有一层灰尘
- **裂痕**: 墙壁和地面有细微裂痕
- **苔藓**: 潮湿的角落长有苔藓
- **烛台**: 墙上的烛台，蜡烛已燃尽

### 光照设计

#### 主光源

- **类型**: 定向光（模拟黄昏阳光）
- **角度**: 从西侧窗户斜射入
- **颜色**: 暖橙色（#FF8C42）
- **强度**: 0.5（微弱）

#### 补光

- **吊灯**: 天花板中央，不发光（关闭状态）
- **符文光**: 齿轮装置的符文，微弱蓝光
- **窗户光**: 从窗户透入的黄昏光

#### 阴影

- **阴影类型**: 软阴影
- **阴影浓度**: 70%
- **环境遮蔽**: 强度 1.0

#### 后期处理

```
Post Process Settings:
├─ Bloom: 0.3 (subtle glow)
├─ Vignette: 0.4 (darken edges)
├─ Color Grading:
│  ├─ Saturation: 0.7 (desaturated)
│  ├─ Contrast: 1.2 (high contrast)
│  └─ Shadows Tint: Blue (#1A2B3C)
├─ Depth of Field: Disabled
└─ Film Grain: 0.2 (subtle texture)
```

### 音效设计

#### 环境音 (循环)

- **风声**: 从窗户缝隙传来的低沉风声
- **时钟滴答**: 远处传来的微弱滴答声（但看不到时钟）
- **木头嘎吱**: 建筑偶尔发出的嘎吱声
- **乌鸦鸣叫**: 远处传来的乌鸦叫声（从顶层）

#### 交互音效

- **脚步声**: 在大理石地面上的脚步声（清脆）
- **开门声**: 沉重的木门声
- **拾取物品**: 轻微的"叮"声
- **阅读日记**: 翻页的"沙沙"声

#### 音乐

- **背景音乐**: 缓慢、低沉的弦乐
- **情绪**: 神秘、忧郁、期待
- **音量**: 很低，几乎像环境音

### 详细布局

#### 2.1 音乐盒 (中央)

**位置**: 房间中央，一个精致的桌子上

**桌子**:
- 类型: 圆形桌子，直径1米
- 材质: 深色木质，桌面有雕刻
- 高度: 0.8米
- 桌腿: 雕刻成藤蔓形状

**音乐盒**:
- 尺寸: 30cm × 20cm × 15cm
- 材质: 红木 + 黄铜装饰
- 外观: 精美的雕刻，顶部有音符图案
- 状态: 初始锁定，需要钥匙
- 细节: 
  - 锁孔在正面，银色
  - 透过缝隙可以看到内部的机械鸟
  - 盒子表面有细微的划痕

**内部（解锁后）**:
- 7个音符按钮: Do, Re, Mi, Fa, Sol, La, Si
- 按钮材质: 象牙白色，上面刻着音符符号
- 机械鸟: 黄铜制，精致，被固定在中央
  - 鸟的翅膀: 可以扇动
  - 鸟的眼睛: 红宝石
  - 鸟的状态: 初始时微微颤抖（痛苦）
- 底部抽屉: 隐藏，完成谜题后打开

**光照**:
- 桌子上方有一盏小吊灯（发出温暖的光）
- 音乐盒在光下闪闪发光

#### 2.2 工作台与甲虫 (南侧)

**工作台**:
- 位置: 南墙边，靠近东侧
- 类型: 实验室工作台
- 尺寸: 2m × 0.8m
- 材质: 木质台面，铁质支架

**工作台上的物品**:
- **手术工具**: 小刀、镊子、放大镜（整齐排列）
  - 小刀: 银色，刀刃锋利
  - 镊子: 黄铜，精密
  - 放大镜: 木柄，玻璃清晰
- **烧杯和试管**: 各种玻璃器皿（空的）
- **笔记本**: 打开着，上面有解剖图
- **蜡烛**: 已燃尽的蜡烛

**死甲虫**:
- 位置: 工作台上，放在一个小盘子里
- 外观: 干枯的黑色甲虫，约5cm长
- 特征: 外壳有不自然的缝隙（暗示可以解剖）
- 状态: 可以拾取和解剖

**工作台下方**:
- 抽屉: 三个抽屉，可以打开（空的）
- 地面: 散落的纸张和工具

#### 2.3 墙上的乐谱 (北墙)

**位置**: 北墙中央，挂在墙上

**乐谱**:
- 类型: 装裱的乐谱
- 尺寸: 60cm × 40cm
- 内容: 一段旋律的音符
  - 音符序列: Mi - Sol - Mi - Do - Re - Mi - Re - Do
- 状态: 从底层看很模糊，需要用望远镜（三层）观察

**装裱**:
- 画框: 简单的木质画框
- 玻璃: 有反光，增加观察难度

#### 2.4 螺旋楼梯 (中央偏东)

**位置**: 与底层楼梯对齐

**楼梯特征**:
- 继续向上延伸到三层
- 栏杆上的藤蔓更密集
- 台阶上有更多的锈迹

#### 2.5 机械装置 (西侧)

**位置**: 西墙边

**装置描述**:
- 一个复杂的机械装置，连接到墙壁
- 功能: 未知（装饰性，暗示这是机械室）
- 外观: 齿轮、杠杆、管道
- 状态: 静止不动

**细节**:
- 齿轮: 大小不一，有锈迹
- 管道: 铜质，有绿色铜锈
- 压力表: 指针指向零

#### 2.6 日记第二页 (东北角)

**位置**: 东北角的一个小架子上

**架子**:
- 类型: 木质书架，三层

**日记**:
- 位置: 书架第二层

### 详细布局

#### 3.1 幻灯机 (西侧)

**位置**: 西墙边，面向东墙

**幻灯机**:
- 类型: 19世纪老式幻灯机
- 尺寸: 50cm × 30cm × 40cm
- 材质: 黄铜 + 木质
- 外观: 
  - 镜头: 大型玻璃镜头，可以调节焦距
  - 光源: 内部有油灯（魔改为永久光源）

- 状态: 初始无幻灯片，插入后激活

**幻灯机底座**:
- 类型: 三脚架
- 材质: 木质，可以调节高度
- 细节: 腿部有雕刻

**焦距调节**:
- 方式: 旋转镜头前的调焦环
- 范围: 0-100%
- 反馈: 投影画面实时变化

#### 3.2 投影墙 (东墙)

**位置**: 东墙，整面墙

**墙面**:
- 材质: 白色石墙（用于投影）
- 尺寸: 4m × 3m
