## 🎬 动画序列建议

### 开场动画流程（约60秒）

```
1. [0-10s] 博物馆外观 - 镜头推进，雾气流动
2. [10-20s] 入口大门 - 门缓缓打开，金光涌出
3. [20-30s] 炼金术士剪影 - 从阴影走出，手持钥匙
4. [30-45s] 中央大厅 - 环视三扇门，每扇门发光
5. [45-60s] 独白："三个展厅，三段时间，一个答案。"
```

### 章节过渡流程（约15秒）

```
1. [0-5s] 符号获得特效 - 符号飞入物品栏
2. [5-10s] 走廊行走 - 镜头跟随，灯光依次点亮
3. [10-15s] 门打开 - 光芒涌出，进入下一章
```

### 回忆过场流程（约30秒）

```
1. [0-5s] 淡入回忆场景，柔焦效果
2. [5-20s] 回忆核心画面，配合独白
3. [20-25s] 情感高潮（音符飞舞/重逢拥抱）
4. [25-30s] 淡出，回到博物馆
```

### 高潮序列流程（约90秒）

```
1. [0-10s] 午夜钟声开始，大钟特写
2. [10-70s] 12次钟声，每次闪回一个月（5秒/次）
3. [70-80s] 三符号飞行融合
4. [80-85s] 永恒之符浮现
5. [85-90s] 天体仪光爆，屏幕全白
```

### 结局动画流程（约60秒）

```
1. [0-10s] 实验室第一人称，独白开始
2. [10-25s] 博物馆消失序列
3. [25-40s] 音乐盒打开，旋律响起
4. [40-50s] 照片上传提示（可选互动）
5. [50-60s] 最终文本，音乐盒关闭，淡出
```

## 💡 创意亮点

### 1. 开场的仪式感

博物馆外观 → 入口 → 大厅的三步递进，营造出"进入神秘世界"的仪式感，让玩家心理上做好准备。

### 2. 走廊的时间暗示

两条走廊的风格差异（狭窄→宽阔，简朴→华丽）暗示着玩家在博物馆中的"成长"和接近真相。

### 3. 回忆的真实感

演唱会和安吉的回忆都使用剪影+柔焦，既保护玩家的想象空间，又让每个人都能代入自己的记忆。

### 4. 高潮的视觉震撼

三符号融合 → 永恒之符 → 天体仪爆发的三重高潮，视觉冲击力层层递进，确保玩家永远记住这一刻。

### 5. 结局的情感升华

从虚拟的博物馆回到真实的实验室，再到邀请玩家放入自己的照片，完美实现了"游戏 → 现实"的情感桥梁。
