## 📐 技术规范

### 文件格式

- **格式**：PNG（带透明通道）
- **色彩模式**：RGBA
- **位深度**：8位/通道

### 分辨率标准

| 资产类型 | 分辨率 | 备注 |
|---------|--------|------|
| 标准场景 | 1920x1080 | 16:9比例 |
| 宽屏场景 | 2560x1080 | 21:9比例（第三章） |
| 物品栏图标 | 512x512 | 正方形 |
| 特殊符号 | 1024x1024 | 高质量符号 |

### 文件命名规范

```
[类别]_[对象]_[状态].png
```

**类别前缀**：
- `BG_` - 场景背景
- `CHAR_` - 角色
- `OBJ_` - 可交互物品
- `ITEM_` - 物品栏图标
- `PAINT_` - 画作
- `INTRO_` - 开场序列
- `TRANS_` - 章节过渡
- `MEMORY_` - 回忆过场
- `CLIMAX_` - 高潮序列
- `ENDING_` - 结局动画
