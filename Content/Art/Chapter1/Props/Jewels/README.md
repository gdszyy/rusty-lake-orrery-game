## 艺术风格

### 锈湖风格特征

所有珠宝资产严格遵循锈湖系列的艺术风格：

- ✅ **版画质感**: 交叉排线阴影，手绘质感
- ✅ **低饱和度色调**: 暖金色、深褐色、幽蓝色为主
- ✅ **强烈光影对比**: 从上方照射的聚光灯效果
- ✅ **透明背景**: PNG格式，无白边
- ✅ **清晰轮廓**: 强烈的黑色描边，易于识别

### 特效设计

#### 闪烁动画原理

1. **帧1（微光）**: 宝石边缘开始出现微弱光晕
2. **帧2（最亮）**: 光芒达到顶峰，带彩虹折射和光线放射
3. **帧3（渐暗）**: 光芒逐渐消退，回到接近基准状态

#### 光效颜色方案

- **钻石**: 白色光 + 彩虹折射（红橙黄绿蓝靛紫）
- **珍珠**: 柔和的奶白色珠光 + 淡粉/淡蓝晕彩
- **红宝石**: 深红色光芒
- **蓝宝石**: 皇家蓝光芒
- **黄金**: 暖金色光晕

## 游戏设计集成

### 谜题流程

根据 `Docs/Chapter1_Design_Optimized.md` 的设计：

1. **触发条件**: 玩家点击卡地亚展柜
2. **动画顺序**: 戒指 → 项链 → 王冠，依次闪烁
3. **音效配合**: 每次闪烁时播放空灵的低语音效
4. **独白播放**: 每件珠宝闪烁时播放对应的独白文本
5. **结束动作**: 三件珠宝说完后，展柜自动打开

### 特效叠加（UE实现）

以下特效将在UE中制作，无需额外资产：

- **冰霜纹理**: 戒指闪烁时，周围出现冰晶纹理
- **锁链纹理**: 项链闪烁时，周围出现锁链幻影
- **枷锁纹理**: 王冠闪烁时，周围出现枷锁幻影

## 使用说明

### UE导入

1. 将所有PNG文件导入UE项目的 `Content/Textures/Chapter1/Jewels/` 目录
2. 为每件珠宝创建材质，启用透明度
3. 创建动画蓝图，设置3帧循环播放

### 动画设置建议

```
Timeline: 0.0s - 0.3s - 0.6s - 0.9s - 1.2s (循环)
Frame:    Base - Glow1 - Glow2 - Glow3 - Base

建议帧率: 每帧0.3秒（约3.3 FPS）
循环模式: Loop
```

### 音效同步

- **戒指**: 清脆的"叮"声 + 冰晶碎裂声
- **项链**: 柔和的"叮"声 + 锁链声
- **王冠**: 宏大的"叮"声 + 金属共鸣
