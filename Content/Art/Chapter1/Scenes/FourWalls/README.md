## 制作方法

### 方法论：完整生成 + 元素消除法

**第一步：生成完整场景**
1. 使用AI图像生成工具（nano banana）
2. 提示词包含：
   - 锈湖风格核心要素（版画质感、交叉排线、低饱和度）
   - 完整场景描述（墙面 + 所有道具）
   - 使用已生成的道具作为参考图
   - 精确的光照描述（聚光灯位置、强度、颜色）
3. 生成1920x1080的完整场景图

**第二步：消除可交互元素**
1. 使用AI图像编辑工具（generate_image_variation）
2. 提示词明确指定要移除的所有道具
3. 要求AI用匹配的背景元素填充：
   - 延伸木质护墙板纹理
   - 延续大理石地板图案
   - 保持光照效果（但照亮空地面）
4. 确保无缝融合，保持交叉排线质感

**优势**:
- ✅ 完整场景和空背景的光照、透视、风格完全一致
- ✅ 空背景的光照"预留"了道具位置
- ✅ 道具可以完美叠加到空背景上
- ✅ 避免单独绘制背景时的不匹配问题

## 游戏集成

### 使用方式

**方案一：分层渲染（推荐）**
```
Layer 1 (底层): 空背景 (BG_Wall*_Empty.png)
Layer 2 (中层): 可交互道具 (OBJ_*.png) - 动态加载/移除
Layer 3 (顶层): UI元素、光效粒子
```

**方案二：状态切换**
```
初始状态: 完整场景 (BG_Wall*_Complete.png)
交互后: 空背景 + 单独道具层
```

### 道具定位参考

使用完整场景图作为道具位置参考：
1. 在UE中加载完整场景图
2. 标记每个道具的Transform（位置、缩放）
3. 替换为空背景 + 独立道具层
4. 微调道具位置确保完美对齐
