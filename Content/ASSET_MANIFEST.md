## 🎬 过场动画 (Cutscenes)

### 1. CUTSCENE_Concert_Full.mp4
- **章节**: 第一章 - 小刚演唱会回忆
- **时长**: ~5秒（视频生成工具限制）
- **描述**: 超现实主义演唱会回忆序列
- **关键场景**:
  - 舞台全景与歌手剪影
  - 人群中牵手的情侣特写
  - 歌手变形为神秘乌鸦
  - 金色音符如雪飘落
  - 时间停滞效果（黑白背景，彩色情侣）
  - 过去之符（羽毛形）浮现
- **色调**: 暖金色与深蓝色
- **用途**: 第一章天平平衡后触发

### 2. CUTSCENE_ItTakesTwo_Full.mp4
- **章节**: 第二章 - 双人成形回忆
- **时长**: ~5秒
- **描述**: 镜像合作与分离的象征序列
- **关键场景**:
  - 破裂镜子与红蓝人偶
  - 同步按下按钮的合作时刻
  - 分离路径的镜像移动
  - 心形平台上的重逢
  - 现在之符（交织花形）浮现
- **色调**: 冷蓝色调与红蓝对比
- **用途**: 第二章镜像谜题完成后触发

### 3. CUTSCENE_Repin_Full.mp4
- **章节**: 第三章 - 列宾艺术展回忆
- **时长**: ~5秒
- **描述**: 艺术创作与流动未来的序列
- **关键场景**:
  - 博物馆墙上的列宾风格画作
  - 空白画布的聚光灯特写
  - 彩虹画笔连接星辰的绘画过程
  - 画作变成流动星空窗户
  - 两个发光人影在星河中共舞
  - 未来之符（彩虹画笔形）浮现
- **色调**: 古典艺术风格转向动态星空
- **用途**: 第三章绘画谜题完成后触发

## 🎨 艺术风格指南

### 色彩方案
- **第一章**: 暖金色 + 深蓝色（演唱会氛围）
- **第二章**: 冷蓝色 + 红蓝对比（镜像与合作）
- **第三章**: 古典棕褐 → 深邃星空蓝 + 金色（艺术转向宇宙）
- **结局**: 深蓝星空 + 温暖金色（永恒与希望）

### 质感要求
- 所有资产使用手绘版画质感
- 维多利亚时代美学
- 铜版画雕刻纹理
- 神秘主义与炼金术符号

### 技术规范
- **视频格式**: MP4, H.264编码
- **图像格式**: PNG, 24位真彩色
- **分辨率**: 1920x1080 (横向), 1024x1024 (方形)
- **透明通道**: 需要抠图的资产使用绿幕背景生成

## 🔧 使用建议

### 在Unreal Engine中使用

1. **导入过场动画**:
   ```
   Content/Movies/ 目录
   使用Media Player和Media Texture播放
   ```

2. **导入VFX**:
   ```
   可转换为序列帧导入Niagara粒子系统
   或直接使用Media Texture播放
   ```

3. **导入静态纹理**:
   ```
   Content/Textures/Ending/ 目录
   直接应用到UI Widget或3D材质
   ```

4. **绿幕抠图处理**:
   ```
   使用Chroma Key材质节点
   或使用外部工具（如After Effects）预处理
   ```

### 优化建议

- **视频压缩**: 根据目标平台调整码率
- **纹理压缩**: 使用DXT5或BC7压缩格式
- **LOD设置**: 为低端设备准备低分辨率版本
- **流式加载**: 大型视频资产使用流式播放
